# Safety E-Stop Monitoring & Control System

A full-stack industrial safety application designed to monitor, manage, and simulate Emergency Stop (E-Stop) operations in manufacturing and industrial environments.

The system provides real-time monitoring of safety events, emergency-stop activation, system status tracking, and event logging through a modern web interface and ASP.NET Core backend services.

---

## Project Overview

Industrial equipment requires immediate shutdown mechanisms to ensure operator safety during hazardous situations.

The Safety E-Stop Monitoring & Control System was developed to simulate and monitor emergency stop events, allowing operators and administrators to visualize safety status, track alarms, and manage critical shutdown operations efficiently.
